# ECHO Frontend

React frontend for the ECHO English Learning Web App.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

Set your API base URL in the hosting environment:
```env
VITE_API_URL=https://your-backend-domain.com
```

3. Build for production:
```bash
npm run build
```

## Features

- 🗣️ AI-powered conversation practice
- 🎬 Movie/series recommendations
- 📚 Grammar lessons and blogs
- 🔐 User authentication
- 📊 Performance reports

## Tech Stack

- React 18
- React Router v6
- Tailwind CSS
- Axios
- Vite
